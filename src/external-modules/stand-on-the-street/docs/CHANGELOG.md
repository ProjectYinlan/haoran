# 当前版本

## 2.1.0 - 2026/3/22

- 新增 6 种道具：咖啡（即时减 CD）、高跟鞋（收入 x2 / CD x1.5）、喇叭（路人 x2）、回旋镖（反弹操伤害）、红包（群友保底）、小药丸（操连击 2-3 次）
- 新增 5 种事件：同行竞争、老板巡街、团购活动、直播翻车、彩票中奖
- 新增 双事件机制：60% 触发首个事件，额外 30% 再触发第二个事件，效果叠加
- 新增 站街 Plan 订阅制（Pro / Max）：CD 期间额外站街，30% 制裁概率，7 天有效期
- 新增 购买卡片、Plan 状态卡片、Plan 订阅卡片模板
- 更新 客流量改为 5 小时滑动窗口（每人每 5h 最多被抽 2 次），不再按日重置
- 更新 每日重置时间统一为凌晨 4:00（Plan 日额度、客流等）
- 更新 操模式不触发事件、不消耗幸运草/护身符
- 更新 回旋镖被动触发时在调用者的结果卡片中显示
- 更新 红包保底群友无视余额/频次限制强制出现
- 更新 结果卡片：道具效果详情、多事件展示、收支分开显示、lucide-react 图标
- 修复 群友显示 0 人但有金额的问题
- 修复 幸运草改为必定触发正面事件（原为正面概率 x2）

## 2.0.0 - 2026/3/22

- 新增 随机事件系统：每次站街有概率触发正面/负面/混合事件
- 新增 道具商店：广告牌、防狼喷雾、保险、贵宾卡、护身符、幸运草
- 新增 背包系统：查看和管理持有道具
- 删除 连续站街功能
- 删除 固定手续费机制（富豪手续费、强制手续费、杨威Buff）
- 更新 手续费由随机事件系统动态替代：税务稽查（替代富豪税）、碰瓷（替代强制手续费）、杨威2.0（替代杨威Buff）
- 更新 强制站街效果改为：跳过 CD + 大幅提高负面事件概率
- 新增 防狼喷雾可阻止被点名
