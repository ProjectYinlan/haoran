# 基本限制
- 存在 `冷却时间`（默认 12h）：未到下次时间会提示，并显示可用的跳过方式。
- 最近 `3` 次站街记录中，最多只能出现 `1` 次无人光顾。
- 路人数量上限 `50`（所有乘数之后）。
- 每日重置时间为 `凌晨 4:00`（包括道具购买次数、付费站街次数、咖啡使用次数）。
- 咖啡副作用独立计时，持续 2 天（不受 4AM 重置影响）。

| `背包` | 查看装备状态、今日可用次数、咖啡副作用等 |
| `站街数据` | 查看个人站街统计 |

# 即买即用道具系统
- 所有道具通过命令参数即买即用，无需提前购买。
- 示例：`站街 广告牌幸运草高跟鞋`、`站街 进攻套餐`、`操 @xxx 小药丸`
- 道具名可以不加空格，系统自动识别。
- 套餐名也可内联使用。
- 每种道具有每日购买上限（4:00 AM 重置）。

## 道具分类

**主动道具**（站街时即时生效）：广告牌、幸运草、高跟鞋、喇叭、保险、红包、贵宾卡、护身符

**操专属道具**（操时即时生效）：小药丸

**被动道具**（站街时装备，被触发时消耗）：防狼喷雾、回旋镖
- `站街 防狼喷雾` 购买并装备，保留到被触发消耗或下次站街替换。
- 被别人操时自动检查装备。

**独立指令**：咖啡（`咖啡` 命令，不与站街绑定）

# 付费站街
- `加钱站街` / `强制站街`：CD 期间可用，每天 1 次，费用 1000。
- 可带道具参数：`加钱站街 广告牌幸运草`。

# 咖啡
- `咖啡` 命令：自动计算剩余 CD 需要几杯（每杯减 2h），一次性扣费并清除 CD。
- 每杯 400 硬币。例如剩余 8h CD → 需要 4 杯 → 花费 1600。
- 如果钱不够喝满，会喝能负担的杯数，CD 部分减少。
- 每天最多使用 1 次咖啡指令（4AM 重置），但一次可以喝多杯。
- 每杯增加 1 层 `咖啡副作用`（每层 +15% 负面事件概率）。
- 副作用持续 2 天，期间所有站街指令（普通/加钱/操）均受影响。到期后自动清除。
- 副作用同时大幅增加第二个事件的触发概率（每层 +20%，基础 30%，上限 95%）。

# 随机事件系统
- 每次站街有 `60%` 基础概率触发随机事件。
- 触发第一个事件后，有额外 `30%` 概率再触发第二个不同事件，效果叠加。
- 事件分为 `正面`、`负面`、`混合` 三类。
- 余额高于 `富豪阈值` 时，负面事件概率额外增加 `50%`。
- 咖啡副作用每层增加 `15%` 负面事件概率。
- `操`（点名）模式不触发事件。

# 经济保护
- 余额 <= 0 的群友被选中时自动白嫖（不扣款）。
- 红包保底群友固定白嫖（score = 0），只计入人头。
- 红包只保底新群友，不会重复保底已有的保底客。
- 高跟鞋 CD 惩罚仅在随机模式生效（点名模式不受影响）。

# 收入计算
- 会根据最近 `5` 次收入计算 `平均收入`。
- 会根据最近 `5` 次站街的 `人均金额` 计算权重：低于 `100` 时，按 `差值 / 100 / 2` 追加到 `基础权重`。
- 付款金额采用加权随机：存在 `基础权重` 与 `高额支付权重`。
- `白嫖` 每出现一次，`高额支付权重` 追加一次。
- 事件效果（收入乘数、客人数乘数等）在基础计算之上叠加。
- 道具效果（广告牌、高跟鞋等）在事件效果之后叠加。

# 随机模式
- 先随机得到 `总人数上限`，再拆分为 `群友数` 与 `路人数`（路人上限 50）。
- 候选 `群友` 需要满足：
  - 余额 `> 0`
  - 近 5 小时被抽次数 `< 2`
- 从候选中按权重抽取 `群友`：余额越多，权重越高，越容易被选中。
- 红包保底的群友无视上述条件强制出现（但白嫖）。
- `群友` 每人得分为 `0~5 x 50`。
- `路人` 每人次独立计算得分：`0~4 x 50`。

# 点名模式
- 每天最多点名 `2 次`（4AM 重置）。
- 一次只能 `点名 1 人`。
- 如果目标装备了 `防狼喷雾`，点名会被阻止（喷雾消耗）。
- 如果目标装备了 `回旋镖`，反弹同等金额给调用者（回旋镖消耗）。
- 目标必须满足：
  - 对方已站过街
  - 余额 `> 0`
  - 近 5 小时被抽次数 `< 2`
- 得分为 `0~11 x 50`。
- 带 `小药丸` 时可连续触发 2-3 次，每次独立随机。
- 点名模式不触发事件。
