# 基本限制
- 存在 `冷却时间`（默认 12h）：未到下次时间且不是 `强制站街` 会直接提示 `下次时间`。
- 若已使用过 `强制站街`，在 `下次时间` 之前无法再次强制。
- 最近 `3` 次站街记录中，最多只能出现 `1` 次无人光顾。
- `连续站街` 已移除。
- 每日重置时间为 `凌晨 4:00`（包括客流量池、Plan 日额度等）。

# 随机事件系统（替代手续费）
- 每次站街有 `60%` 基础概率触发随机事件。
- 触发第一个事件后，有额外 `30%` 概率再触发第二个不同事件，效果叠加。
- 事件分为 `正面`、`负面`、`混合` 三类。
- 余额高于 `富豪阈值` 时，负面事件概率额外增加 `50%`。
- `强制站街` 时，负面事件概率额外增加 `75%`。
- `操`（点名）模式不触发事件。

## 正面事件
| 事件 | 效果 |
|------|------|
| 大富翁路过 | 额外获得 `500~1000` 固定收入 |
| 网红效应 | 群友数量 x2 |
| 双倍快乐 | 本次所有收入 x2 |
| 贵人相助 | 冷却时间减半 |
| 老板巡街 | 额外获得 `200~800` 固定收入，CD x0.7 |
| 彩票中奖 | 额外获得余额 10%（上限 50000）的固定收入 |

## 负面事件
| 事件 | 效果 |
|------|------|
| 城管来了 | 本次收入归零 |
| 碰瓷 | 被扣除 `平均收入 x 0.3~0.8` 的金额 |
| 暴雨 | 客人数量减半 |
| 税务稽查 | 仅余额 > 富豪阈值时出现，税率 `5%~20%` |
| 杨威2.0 | 冷却时间 x3 |
| 同行竞争 | 收入 x0.3 |

## 混合事件
| 事件 | 效果 |
|------|------|
| 打折日 | 路人数量 x3，每人只付 50 |
| 神秘客人 | 只来 1 位客人，出手阔绑或一毛不拔 |
| 团购活动 | 群友数量 x3，每人只付 50 |
| 直播翻车 | 路人数量 x5，但收入 x0.2 |

# 道具商店
- 使用 `站街商店` 查看可购买的道具。
- 使用 `购买 <道具名> [数量]` 购买道具。
- 使用 `背包` 查看持有的道具。

# 站街 Plan（订阅制）
在 CD 期间提供额外站街次数，不影响基础 CD 和贵人/杨威等事件。

- `站街plan sub pro/max`：订阅 Plan。
- `站街plan status`：查看当前 Plan 状态。
- `站街plan unsub`：退订 Plan，按剩余周次数比例 x60% 退款。

| | Plan Pro | Plan Max |
|------|----------|----------|
| 价格 | 1500 | 2500 |
| 日额度 | 2 次 | 3 次 |
| 周额度 | 10 次 | 15 次 |
| 有效期 | 7 天 | 7 天 |
| 制裁概率 | 30% / 次 | 30% / 次 |
| 封禁时长 | 1 天 | 1 天 |

- 每次使用 Plan 站街后，有 `30%` 概率被制裁（Plan 销毁），按剩余周额度比例 x60% 退款。
- 被制裁后需等 1 天才能重新订阅。
- 日额度在每日 `4:00 AM` 重置。
- 订阅不被制裁最多持续 7 天，到期后可重新订阅。

# 收入计算
- 会根据最近 `5` 次收入计算 `平均收入`。
- 会根据最近 `5` 次站街的 `人均金额` 计算权重：低于 `100` 时，按 `差值 / 100 / 2` 追加到 `基础权重`。
- 付款金额采用加权随机：存在 `基础权重` 与 `高额支付权重`。
- `白嫖` 每出现一次，`高额支付权重` 追加一次。
- 事件效果（收入乘数、客人数乘数等）在基础计算之上叠加。
- 道具效果（广告牌、保险等）在事件效果之后叠加。

# 随机模式
- 先随机得到 `总人数上限`，再拆分为 `群友数` 与 `路人数`。
- 候选 `群友` 需要满足：
  - 余额 `> 0`
  - 近 5 小时被抽次数 `< 2`
- 从候选中按权重抽取 `群友`：余额越多，权重越高，越容易被选中。
- 红包保底的群友无视上述条件强制出现。
- `群友` 每人得分为 `0~5 x 50`。
- `路人` 每人次独立计算得分：`0~4 x 50`。

# 点名模式
- 一次只能 `点名 1 人`，否则会提示。
- 如果目标持有 `防狼喷雾`，点名会被阻止。
- 如果目标持有 `回旋镖`，反弹同等金额给调用者。
- 目标必须满足：
  - 对方已站过街
  - 余额 `> 0`
  - 近 5 小时被抽次数 `< 2`
- 得分为 `0~11 x 50`。
- 持有 `小药丸` 时可连续触发 2-3 次，每次独立随机。
- 点名模式不触发事件，不消耗 `幸运草` / `护身符`。
