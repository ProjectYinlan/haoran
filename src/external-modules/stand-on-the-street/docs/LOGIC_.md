# 基本限制
- 存在 `冷却时间`：未到下次时间且不是 `强制站街` 会直接提示 `下次时间`。
- 若已使用过 `强制站街`，在 `下次时间` 之前无法再次强制。
- 最近 `3` 次站街记录中，最多只能出现 `1` 次无人光顾。
- `连续站街` 已移除。

# 随机事件系统（替代手续费）
- 每次站街有 `40%` 基础概率触发随机事件。
- 事件分为 `正面`、`负面`、`混合` 三类。
- 余额高于 `富豪阈值` 时，负面事件概率额外增加 `50%`。
- `强制站街` 时，负面事件概率额外增加 `75%`。

## 正面事件
- **大富翁路过**：额外获得 `500~1000` 固定收入。
- **网红效应**：群友数量上限翻倍。
- **双倍快乐**：本次所有收入 `x2`。
- **贵人相助**：冷却时间减半。

## 负面事件
- **城管来了**：本次收入归零。
- **碰瓷**：被扣除 `平均收入 x 0.3~0.8` 的金额。
- **暴雨**：客人数量减半。
- **税务稽查**：仅在余额 `>` 富豪阈值时出现，税率随余额动态增长 `5%~20%`。
- **杨威2.0**：冷却时间直接 `x3`。

## 混合事件
- **打折日**：路人数量 `x3`，但每人只付 `50`。
- **神秘客人**：只来 `1` 个客人，付费范围 `0~20 x 50`。

# 道具商店
- 使用 `站街商店` 查看可购买的道具。
- 使用 `购买 <道具名> [数量]` 购买道具。
- 使用 `背包` 查看持有的道具。
- 道具在下次站街时自动生效（无需手动使用），消耗后扣减。

# 收入计算
- 会根据最近 `5` 次收入计算 `平均收入`。
- 会根据最近 `5` 次站街的 `人均金额` 计算权重：低于 `100` 时，按 `差值 / 100 / 2` 追加到 `基础权重`。
- 付款金额采用加权随机：存在 `基础权重` 与 `高额支付权重`。
- `白嫖` 每出现一次，`高额支付权重` 追加一次。
- 事件效果（收入乘数、客人数乘数等）在基础计算之上叠加。
- 道具效果（广告牌、保险等）在事件效果之后叠加。

# 随机模式
- 先随机得到 `总人数上限`，再拆分为 `群友数` 与 `路人数`。
- 候选 `群友` 需要满足：
  - 余额 `> 0`
  - 当天被榨次数 `< 2`
- 从候选中按权重抽取 `群友`：余额越多，权重越高，越容易被选中。
- `群友` 每人得分为 `0~5 * 50`。
- `路人` 每人次独立计算得分：`0~4 * 50`。

# 点名模式
- 一次只能 `点名 1 人`，否则会提示。
- 如果目标持有 `防狼喷雾`，点名会被阻止。
- 目标必须满足：
  - 对方已站过街
  - 余额 `> 0`
  - 当天被榨次数 `< 2`
- 得分为 `0~11 * 50`。
